## 在每个树行中找最大值

**原链接**：<https://leetcode-cn.com/problems/find-largest-value-in-each-tree-row/>

#### 树[Tree]    深度优先搜索[Depth-first Search]    广度优先搜索[Breadth-first Search]    

##### 难度：**`Medium`**

----- 
<p>You need to find the largest value in each row of a binary tree.</p>

<p><b>Example:</b><br />
<pre>
<b>Input:</b> 

          1
         / \
        3   2
       / \   \  
      5   3   9 

<b>Output:</b> [1, 3, 9]
</pre>
</p>


----- 
<p>您需要在二叉树的每一行中找到最大的值。</p>

<p><strong>示例：</strong></p>

<pre>
<strong>输入:</strong> 

          1
         / \
        3   2
       / \   \  
      5   3   9 

<strong>输出:</strong> [1, 3, 9]
</pre>
